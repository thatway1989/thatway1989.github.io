# 1. 关于博主:Thatway
详细内容参考：[闲聊-关于公众号改名《那路谈OS与SoC嵌入式软件》](https://mp.weixin.qq.com/s/qm_iMw78kztYmPDICYLU5A) 这里摘抄一部分。

**thatway由来：**

笔者原创用的网名**thatway**，时间拉回到**2005**年左右，当时就开始听后街男孩的英文歌曲，很有旋律。然后**2007**年有了QQ账号就需要起一个网名，时髦点必须搞个英文的，去哪里找呢？寓意：**遵从自己的内心的想法去做**，恰逢又听后街男孩的这支歌曲：**I want it that way**，抛开这是首分手情歌不谈，字面意思不就是：***我想让它那样***。

>名字也算是初心的一种体现，Thatway表达的是：
>
>***人生路认定方向一往直前，坚定的走下去。***

另外此时也开始写一些程序，**运行结果也必须是心中所想**啊。在此也祝愿看到thatway文章的程序员：**写的程序都能运行出自己想要的结果，人生路也能如愿而走**。

**thatway**这个网名从QQ再到微信再到公众号就这么用了**近20年**，使用Thatway1989是加上笔者的出生年份1989。

# 2. 关于公众号:那路谈OS与SoC嵌入式软件
1. **那路**：大概10年前，在西安进山玩**秦岭脚下丰裕口**有个出名的馆子叫《**那家**》很有名，在北京经常去外面吃饭的朋友有没有见过《**那家小馆**》，照葫芦画瓢**thatway**翻译不就是“**那路**”，就是它了。虽然充满了chinglish，但是作为国人应该可以能**看懂**吧。

2. **OS与SoC**：目前本博客专注于汽车软件软件底层的技术，基本就是**OS操作系统和SoC芯片硬件设计**相关的东西了，这也是目前的兴趣所在，但是这些只占工作一来技术的一小部分，后面有机会了可以分享点**跨界的程序员技术**，都开开眼。

3. **嵌入式软件**：回想在学校及工作以来的历程，其实一直都跟**嵌入式**密不可分，做**网络设备、通信设备、终端、芯片、安卓、汽车软件、AI**等。

综上，公众名字修改为：<u>***《那路谈OS与SoC嵌入式软件》***</u>

>公众号初心：学习老美的技术，帮助国人。即使有商业化也不会影响大家的学习阅读，本公众号也不搞付费阅读，卖课等影响大家学习的事，所有的知识不用付一分钱都可以看到。
